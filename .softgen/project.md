# Multi-Tenant Ecommerce Platform

## Vision
Nền tảng thương mại điện tử đa cửa hàng — mỗi shop có storefront riêng và admin dashboard quản lý sản phẩm, đơn hàng, bài viết, danh mục. Frontend-only với mock data. UI tiếng Việt.

## Design
- `--primary: 263 84% 58% (electric violet)`
- `--accent: 25 95% 53% (energetic orange)`
- `--background: 0 0% 98% (near-white)`
- `--foreground: 240 6% 10% (deep slate)`
- `--muted: 240 5% 96% (light gray)`
- `--card: 0 0% 100% (white)`
- Heading font: Montserrat (700, 800)
- Body font: Be Vietnam Pro (400, 500, 600, 700)
- Style: Clean, vibrant, modern — rounded-xl, subtle shadows, gradient accents

## Features
- Platform landing page showcasing featured shops
- Storefront per shop: product grid, product detail, cart
- Admin dashboard: overview stats, product CRUD, category CRUD, order management, post management, shop settings
- Responsive desktop + mobile optimized
- All UI in Vietnamese
